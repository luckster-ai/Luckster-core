# JOTI Platform

A modern digital yoga platform for experiencing Kundalini Yoga.

JOTI is designed to make Kundalini Yoga simple, approachable, and enjoyable.
Rather than overwhelming learners with information, JOTI focuses on helping people experience yoga through guided video practices.

The platform is built as a responsive web application, allowing members to access their practice anytime from desktop, tablet, or mobile devices through a single website.

---

## Vision

Create an accessible digital home for Kundalini Yoga where anyone can begin, continue, and enjoy their yoga journey.

Technology should quietly support the experience, allowing practitioners to focus on yoga rather than the platform itself.

---

## Current Goals

The current development focuses on building a simple and reliable MVP.

Core features include:

- Foundation videos
- Module Library
- Membership system
- Responsive website
- PWA-ready experience
- AI-assisted development workflow

Future features such as music, books, community, and AI services will be added after the core platform is stable.

## Responsive First

JOTI is designed as a responsive web application.

The same React codebase supports:

- Desktop
- Tablet
- Mobile

The platform is also designed to become PWA-ready, allowing users to install it on their devices without requiring native iOS or Android applications.
