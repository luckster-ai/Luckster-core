# JOTI Terminology

## Purpose

本文件定義 JOTI 使用的專有名詞、翻譯原則與使用規範。

目的不是翻譯所有昆達里尼瑜伽術語，而是建立一套一致、容易理解且符合 JOTI 品牌理念的語言系統。

所有網站、課程、AI、文件與未來產品，皆以本文件為共同標準。

---

# Translation Philosophy

JOTI 尊重昆達里尼瑜伽的原始術語。

然而，對於初學者而言，直接使用大量梵文名稱容易增加學習門檻。

因此 JOTI 採用：

**先理解，再認識原始名稱。**

網站優先使用容易理解的中文名稱。

在適當時機，再逐步介紹英文或梵文名稱，幫助學員自然建立對原始術語的認識。

---

# Core Terminology

## Foundation（新手基礎）

學習所有正式練習前需要建立的基礎知識、技能與練習準備。

例如：

* 基礎知識
* 呼吸基礎 - 深長呼吸、火呼吸、屏息
* 調頻和結尾唱誦
* 身體鎖
* 手印
* 坐姿
* 練習環境
* 練習裝備
* 常見問題

Foundation 的目的不是完成課程，而是建立基礎能力。

---

## Module（課程元件）

組成一堂完整 Practice 的最小單位。

Module 是網站架構中的概念，並非昆達里尼瑜伽的正式術語。

---

## Practice（完整練習）

由多個 Module 組合而成的一堂完整課程。

例如：

* 晨間能量
* 午間冥想
* 晚間放鬆
* 睡前練習

Practice 是學員主要進行的練習形式。

---

## Kriya

Kriya 為昆達里尼瑜伽的重要術語。

意指一組具有特定目的的練習，由姿勢、呼吸、專注、聲音等元素共同組成，以引導特定身心狀態的產生。

JOTI 不直接將 Kriya 翻譯為「動作」或「體式」。

依照不同情境，可使用：

* 身體練習（Body Practice）
* 呼吸練習（Breath Practice）
* 冥想練習（Meditation Practice）

並在適當時機介紹 Kriya 一詞，協助學員逐步理解其完整涵義。

---

# Naming Principle

所有課程命名應優先描述：

* 練習目的
* 練習體驗
* 練習情境

而不是大量使用難以理解的專有名詞。

例如：

✓ 晨間能量

✓ 睡前放鬆

✓ 壓力釋放

✓ 深度放鬆

而不是：

△ Kriya 3

△ Set A

△ Meditation 7

除非課程本身已有固定且廣泛使用的正式名稱。
