# Module Library

## Purpose

Module 是組成 Practice 的基本單位（Building Block）。

每一堂 Practice 都是由一個或多個 Module 所組成。

每個 Module 都有明確的目的、類型與規則，可由官方建立，也可作為 AI 組課與學員自由組課的基礎元件。

Module 並不等於一支影片。

一個 Module 可以對應一支或多支影片，而影片則是 Module 的內容載體。

---

# Module Categories

目前 JOTI 將 Module 分為六大類：

## 1. 調頻（Tuning In）

Purpose：

建立練習空間，調整身心狀態，正式開始 Practice。

通常包含：

* 開始調頻（Adi Mantra）

每堂 Practice 必須包含一個調頻 Module。

---

## 2. 熱身（Warm Up）

Purpose：

喚醒身體、調整呼吸、準備進入主要練習。

可能包含：

* 呼吸熱身
* 貓牛式
* 脊椎暖身
* 關節活動
* 其他暖身動作

部分身體動作序列已包含完整熱身，因此可省略獨立熱身 Module。

---

## 3. 身體動作序列（Physical Sequence）

Purpose：

透過身體姿勢、呼吸與節奏的結合，完成主要練習內容。

此類 Module 對應昆達里尼瑜伽中的 Physical Kriya。

例如：

* 清晨薩達那
* 心輪練習
* 消化系統練習
* 脊椎能量練習

---

## 4. 放鬆（Relaxation）

Purpose：

讓身體吸收前面練習所帶來的影響，恢復平衡。

通常包含：

* 大休息（Savasana）
* 引導放鬆

部分短時間 Practice 可省略。

---

## 5. 冥想（Meditation）

Purpose：

透過呼吸、唱誦、專注或靜心，整合整堂 Practice。

可能包含：

* 呼吸冥想
* 唱誦冥想
* 靜心冥想

此類 Module 對應昆達里尼瑜伽中的 Meditation Kriya。

大多數 Practice 建議包含至少一個 Meditation Module。

---

## 6. 結尾（Closing）

Purpose：

完成整堂 Practice，回到日常生活。

通常包含：

* Long Time Sun
* Sat Nam

每堂 Practice 必須包含一個 Closing Module。

---

# Module Rules

建立 Practice 時，應遵循以下原則：

* 每堂 Practice 必須包含調頻與結尾。
* 冥想通常作為主要整合階段，建議保留。
* 熱身可依主要身體動作序列調整或省略。
* 放鬆可依課程時間調整。
* 避免相同功能的 Module 重複出現。
* 保持課程流程自然且符合練習目的。

---

# Module Attributes

每個 Module 建議具備以下資訊：

* Title（名稱）
* Category（分類）
* Purpose（目的）
* Description（簡介）
* Duration（建議時間）
* Level（適合程度）
* Video Reference（對應影片）
* Tags（標籤）

---

# Relationship

Foundation 建立基本能力。

Module 是組成 Practice 的基本元件。

Practice 則由一個或多個 Module 所組成，並透過影片提供實際教學與練習內容。
