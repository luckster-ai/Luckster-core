# Module Library

## Purpose

Module 是組成 Practice 的基本單位（Building Block）。

每一堂 Practice 都是由一個或多個 Module 所組成。

每個 Module 都有明確的目的、類型與規則，可由官方建立，也可作為 AI 組課與學員自由組課的基礎元件。

Module 並不等於一支影片。

一個 Module 可以對應一支或多支影片，而影片則是 Module 的內容載體。

---

# Module Categories

目前 JOTI 將 Module 分為六大類：

## 1. 調頻（Tuning In）

Purpose：

建立練習空間，調整身心狀態，正式開始 Practice。

通常包含：

* 調頻（Adi Mantra）
* 調頻（Adi Mantra + 保護咒）

每堂 Practice 必須包含一個調頻 Module。

---

## 2. 熱身（Warm Up）

Purpose：

喚醒身體、調整呼吸、準備進入主要練習。

可能包含：

* 呼吸熱身
* 貓牛式
* 脊椎暖身
* 關節活動
* 其他暖身動作

部分身體動作序列已包含完整熱身，因此可省略獨立熱身 Module。

---

## 3. 身體動作序列（Physical Sequence）

Purpose：

透過身體姿勢、呼吸與節奏的結合，完成主要練習內容。

此類 Module 對應昆達里尼瑜伽中的 Physical Kriya。

例如：

* 清晨薩達那
* 心輪練習
* 消化系統練習
* 脊椎能量練習

---

## 4. 放鬆（Relaxation）

Purpose：

讓身體吸收前面練習所帶來的影響，恢復平衡。

通常包含：

* 大休息（Savasana）
* 引導放鬆

部分短時間 Practice 可省略。

---

## 5. 冥想（Meditation）

Purpose：

透過呼吸、唱誦、專注或靜心，整合整堂 Practice。

可能包含：

* 呼吸冥想
* 唱誦冥想
* 靜心冥想

此類 Module 對應昆達里尼瑜伽中的 Meditation Kriya。

大多數 Practice 建議包含至少一個 Meditation Module。

---

## 6. 結尾（Closing）

Purpose：

完成整堂 Practice，回到日常生活。

通常包含：

* Long Time Sun
* Sat Nam

每堂 Practice 必須包含一個 Closing Module。

---

# Module Rules

JOTI 目前將 Practice 分為兩種主要形式。

---

## Type A：完整 Practice（Full Practice）

完整 Practice 為 JOTI 最主要的課程形式。

一般包含：

1. 調頻（Tuning In）
2. 熱身（Warm Up，可依需要）
3. 身體動作序列（Physical Sequence，可依課程需要）
4. 放鬆（Relaxation）
5. 冥想（Meditation）
6. 結尾（Closing）

### 規則

* 每堂課必須包含調頻與結尾。
* 冥想為整合整堂課的重要環節，建議保留。
* 若包含身體動作序列，應安排放鬆，協助身體吸收練習效果。
* 若沒有身體動作序列，可依課程設計決定是否安排放鬆。
* 熱身可獨立安排，也可已包含於身體動作序列中。

---

## Type B：Meditation Practice（冥想 Practice）

主要作為午間、夜間或短時間練習。

課程時間通常為 30 分鐘以內。

一般包含：

1. 調頻（Tuning In）
2. 熱身（Warm Up）
3. 冥想（Meditation）
4. 結尾（Closing）

### 規則

* 不包含身體動作序列（Physical Sequence）。
* 通常不安排放鬆（Relaxation）。
* 熱身以短時間的脊椎活動、呼吸準備與身體喚醒為主，協助學員穩定坐姿並進入冥想狀態。
* 適合午間、夜間或日常維持練習。

常見熱身包括：

* 脊椎彎曲（Spinal Flex）
* 頸部轉動（Neck Rolls）
* 消除小我（Ego Eradicator）
* 拜日式
* 貓牛式
* 脊椎扭轉
* 生命神經伸展
* 水瓶行軍
* 起身前屈

可依冥想內容與課程目的選擇適合的熱身 Module。

---

AI 建立 Practice 或學員自由組課時，皆應依照以上規則安排 Module，以維持課程完整性與安全性。


---

# Module Attributes

每個 Module 建議具備以下資訊：

* Title（名稱）
* Category（分類）
* Purpose（目的）
* Description（簡介）
* Duration（建議時間）
* Level（適合程度）
* Video Reference（對應影片）
* Tags（標籤）

---

# Relationship

Foundation 建立基本能力。

Module 是組成 Practice 的基本元件。

Practice 則由一個或多個 Module 所組成，並透過影片提供實際教學與練習內容。
