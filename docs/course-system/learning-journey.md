# Learning Journey

## Purpose

本文件描述 JOTI 學員的完整學習旅程（Learning Journey）。

JOTI 並不是以完成課程為目標，而是希望陪伴學員建立長期且穩定的昆達里尼瑜伽練習習慣。

因此，本文件將作為網站、AI 與未來產品設計的重要依據。

---

# Learning Philosophy

JOTI 相信：

真正的學習，來自持續的練習，而不是觀看更多影片。

每一次呼吸、每一次練習、每一次回到墊子上，都在慢慢建立新的身心習慣。

網站的目標不是讓學員快速完成所有課程，而是願意一次又一次回到自己的練習。

---

# Stage 1：Discover（認識）

第一次接觸 JOTI。

學員開始了解：

* 什麼是昆達里尼瑜伽
* JOTI 的理念
* 昆達里尼 ABC 學習方式
* 是否適合自己的需求

此階段的目標是降低進入門檻，而不是立即開始大量練習。

---

# Stage 2：Foundation（建立基礎）

開始建立正式練習所需的基礎能力。

包括：

* 基礎知識
* 呼吸基礎
* 調頻與結尾唱誦
* 身體鎖
* 手印
* 坐姿
* 練習環境
* 練習裝備
* 常見問題

Foundation 以老師示範與講解為主。

學員可跟著練習，但主要目的是理解方法，而不是完成一堂完整課程。

---

# Stage 3：Practice（開始練習）

完成 Foundation 後，學員開始進入完整 Practice。

Practice 由多個 Module 組合而成。

例如：

* 調頻
* 熱身（可依課程需要省略）
* 身體練習
* 大休息
* 呼吸或冥想練習
* 結尾唱誦

不同 Practice 可依練習目的、時間長度與身心狀態進行設計。

---

# Stage 4：Personal Practice（建立自己的練習）

當學員逐漸熟悉課程後，可以依照自己的需求選擇不同方式：

* 官方推薦 Practice
* AI 協助建立 Practice
* 自由組合 Module 建立 Practice

JOTI 尊重每位學員不同的生活節奏與身心需求，鼓勵建立屬於自己的練習方式。

---

# Stage 5：Long-term Practice（長期陪伴）

JOTI 鼓勵學員建立規律且長期的練習，而非追求完成更多課程。

昆達里尼瑜伽傳統相信：

* 40 天建立新的習慣
* 90 天穩固新的習慣
* 120 天成為新的自己
* 1000 天深度融入生活

JOTI 尊重這項傳統，同時尊重每位學員目前的練習狀態。

網站將鼓勵持續回到練習，而不是要求所有人完成固定挑戰。

---

# AI Companion

Luckster AI 是陪伴者，而不是老師的替代者。

AI 可以：

* 根據學員需求推薦 Practice
* 協助建立完整 Practice
* 避免 Module 重複
* 提醒持續練習
* 記錄學習歷程
* 協助建立長期習慣

AI 的目標不是安排更多課程，而是陪伴學員持續回到自己的練習。

這也是 JOTI 希望陪伴每位學員走上的學習旅程。
