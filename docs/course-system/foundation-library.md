# Foundation Library

## Purpose

Foundation 是學員開始正式 Practice 前需要建立的基本能力。

Foundation 的目的不是完成課程，而是建立安全且穩定的練習基礎。

每個 Foundation 完成後，都會記錄完成狀態，作為後續 Practice 與 AI 推薦的依據。

---

# Completion Status

每個 Foundation 都具有完成狀態。

例如：

* Not Started
* In Progress
* Completed
* Self Verified（已自行確認具備能力）

Self Verified 提供已有昆達里尼瑜伽經驗的學員使用。

學員可以自行標記已具備能力，而不需要重新觀看所有 Foundation 內容。

# Foundation Rules

Foundation 本身通常不需要任何 Prerequisites。

完成 Foundation 後，將成為 Practice 的先備能力（Prerequisites）。

網站與 AI 可以依據完成狀態：

* 推薦適合的 Practice
* 提醒尚未完成的重要 Foundation
* 協助建立學習計畫
* 記錄長期練習歷程
