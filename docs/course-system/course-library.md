# Course 01

課程名稱：

昆達里尼 ABC 入門課

目標：

第一次接觸昆達里尼瑜伽的新學員。

時間：

約 45 分鐘

課程結構：

1. 黃金鏈接與保護咒

2. 深長呼吸

3. 提升的奎亞

4. 深度放鬆

5. 生命力冥想（3分鐘）

6. 永恆的陽光

主要效益：

* 建立練習習慣
* 認識 ABC 系統
* 穩定身心

適合：

完全初學者

→ Relaxation

→ Ending

自動推薦或生成個人化課程。
