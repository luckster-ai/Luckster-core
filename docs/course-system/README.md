# JOTI Course System

## Purpose

JOTI Course System 是 JOTI 昆達里尼 ABC 瑜伽的核心教學架構。

它定義了課程如何設計、如何組成、如何學習，以及 AI 如何協助學員建立穩定的練習習慣。

本文件作為整個課程系統的最高規格文件（System Specification）。

詳細內容則分別記錄於：

* foundation-library.md

# Learning Philosophy

JOTI 並不是一個以「完成所有課程」為目標的線上學習平台。

昆達里尼瑜伽的核心並非累積知識，而是透過持續而規律的練習，逐漸建立新的身心習慣。

因此，學員可能會反覆練習同一支影片、同一套練習，而不是不斷追求新的內容。

JOTI 鼓勵的是：

學會方法 → 持續練習 → 建立習慣 → 融入生活。

---

昆達里尼瑜伽傳統中相信：

* 40 天建立新的習慣
* 90 天穩固新的習慣
* 120 天成為新的自己
* 1000 天完全掌握新的生命模式

JOTI 尊重這項傳統，也尊重每位學員目前的練習狀態。

網站將鼓勵持續練習，而不是要求所有學員一開始就完成長期挑戰。

---

# Course Architecture

JOTI 課程系統由三個主要層級組成：

Foundation
↓

Module
↓

Practice

Foundation 建立學員的基本能力。

Module 為組成課程的基本元件。

Practice 則是學員實際進行的一堂完整練習。

---

# AI Philosophy

Luckster AI 的角色並不是取代老師。

AI 的任務包括：

* 協助學員找到適合自己的練習。
* 根據課程規則協助組成完整 Practice。
* 避免模組重複或不合理搭配。
* 協助建立穩定而長期的練習習慣。

最終，希望讓每位學員都能依照自己的身心狀態，持續、安全且自在地練習昆達里尼瑜伽。
