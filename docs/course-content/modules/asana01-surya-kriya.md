## Description

Surya Kriya 以太陽能量（Surya）的品質為主題，透過一系列體式與呼吸的結合，系統性地刺激陽性的普拉那能量與昆達里尼能量。這套奎亞有助於提升活力、淨化身心，並為日常薩達那練習注入穩定而充沛的能量。

## Learning Outcomes

完成本 Module 後，學員將有機會：

- 體驗陽性普拉那能量與活力的提升。
- 感受消化、專注與行動力的增強。
- 在日常薩達那中培養更充沛的體力與多工能力。

---

## Notes

這個奎亞以太陽能量的名字命名，稱為 Surya。當太陽能量充盈時，人會精力充沛、善於表達、開朗外向且熱情洋溢。這是一種具有淨化功能的能量。此練習有助於減輕體重、促進消化、提升思維清晰度、分析力與行動力。建議在日常薩達那中不時加入本奎亞，以增強體力與同時處理多項事務的能力。
