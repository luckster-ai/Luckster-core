## Description

深長呼吸是昆達里尼 ABC 瑜伽中最重要的基礎呼吸方式之一，也是大多數 Practice 的基本呼吸模式。

本 Foundation 將介紹深長呼吸的原理、練習方式，以及在正式課程中的應用。

## Learning Outcomes

完成本 Foundation 後，學員應能：

- 理解深長呼吸的目的與重要性。
- 正確完成深長呼吸。
- 在正式 Practice 中自然運用深長呼吸。
