## Description

身體鎖是昆達里尼瑜伽的重要能量技巧，包括根鎖、橫膈膜鎖與頸鎖。正確掌握身體鎖，有助於穩定姿勢並引導能量流動。

本 Foundation 將介紹三大身體鎖的目的、基本操作與安全原則。

---

## Prerequisites

- Long Deep Breathing（FD001）

## Learning Outcomes

完成本 Foundation 後，學員應能：

- 了解三大身體鎖的功能。
- 在安全與放鬆的狀態下練習基本身體鎖。
- 在後續 Practice 中辨識並運用適當的身體鎖。
