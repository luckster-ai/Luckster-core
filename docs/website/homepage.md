# ABC 系統介紹

標題：

什麼是昆達里尼 ABC 瑜伽？

說明：

ABC 代表：

A = Asana（體式）

建立穩定的身體基礎。

B = Breath（呼吸）

學習管理能量與情緒。

C = Chant（唱誦）

透過聲音與冥想培養覺察。

結尾：

以簡單易懂的方式，
帶領初學者進入昆達里尼瑜伽。
